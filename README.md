# Loupe

Open-source observability and replay for LLM agents.

Instrument your agent with three lines of Python, see every trace in a dashboard, then replay any trace with a different prompt or model and compare outputs side-by-side.

> Status: early development. See [CLAUDE.md](./CLAUDE.md) for the build checklist.

## Repo layout

- `sdk/` — Python SDK, published to PyPI as `loupe-sdk`
- `server/` — FastAPI backend
- `dashboard/` — Next.js 14 frontend
